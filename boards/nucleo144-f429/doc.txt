## Overview

Cortex-M4
STM32F429ZI microcontroller with 128Kb of SRAM and 1Mb of ROM Flash.

## Flashing the device

The ST Nucleo-F429ZI board includes an on-board ST-LINK V2-1 programmer. The
easiest way to program the board is to use OpenOCD. Once you have installed
OpenOCD (look [here](https://github.com/RIOT-OS/RIOT/wiki/OpenOCD) for
installation instructions), you can flash the board simply by typing

```
make BOARD=nucleo144-f429 flash
```
and debug via GDB by simply typing
```
make BOARD=nucleo144-f429 debug
```

## Supported Toolchains

For using the ST Nucleo144-F429 board we strongly recommend the usage of the
[GNU Tools for ARM Embedded Processors](https://launchpad.net/gcc-arm-embedded)
toolchain.
