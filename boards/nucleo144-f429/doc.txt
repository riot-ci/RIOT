/**
 * @defgroup    boards_nucleo144-f429 STM32 Nucleo144-F429
 * @ingroup     boards_common_nucleo144
 * @brief       Support for the STM32 Nucleo144-F429
 *
 * ## Overview
 *
 * Not yet available upstream, see [PR 6594](https://github.com/RIOT-OS/RIOT/pull/6594)
 *
 * The Nucleo144-F429 is a board from ST's Nucleo family supporting a ARM
 * Cortex-M4
 * STM32F429ZI microcontroller with 128Kb of SRAM and 1Mb of ROM Flash.
 *
 * ## Hardware
 *
 * ![Nucleo144 F429](https://raw.githubusercontent.com/wiki/RIOT-OS/RIOT/images/nucleo144-board.png)
 *
 * ### MCU
 *
 * | MCU        | STM32F429ZI       |
 * |:------------- |:--------------------- |
 * | Family | ARM Cortex-M4     |
 * | Vendor | ST Microelectronics   |
 * | RAM        | 256Kb |
 * | Flash      | 2Mb               |
 * | Frequency  | up to 180MHz |
 * | FPU        | yes               |
 * | Timers | 17 (2x watchdog, 1 SysTick, 12x 16-bit, 2x 32-bit)    |
 * | ADCs       | 3x 12-bit         |
 * | UARTs      | 8                 |
 * | SPIs       | 6                 |
 * | I2Cs       | 3                 |
 * | RTC        | 1                 |
 * | Vcc        | 2.0V - 3.6V           |
 * | Datasheet  | [Datasheet](http://www.st.com/resource/en/datasheet/stm32f429zi.pdf) |
 * | Reference Manual | [Reference Manual](http://www.st.com/resource/en/reference_manual/dm00031020.pdf) |
 * | Programming Manual | [Programming Manual](http://www.st.com/resource/en/programming_manual/dm00046982.pdf) |
 * | Board Manual   | [Board Manual](http://www.st.com/resource/en/user_manual/dm00244518.pdf)|
 *
 * ## Implementation Status
 *
 * | Device | ID        | Supported | Comments  |
 * |:------------- |:------------- |:------------- |:------------- |
 * | MCU        | STM32F429ZI   | partly    | Energy saving modes not fully utilized |
 * | Low-level driver | GPIO    | yes       | |
 * |        | PWM       | yes (9 pins available)    |  |
 * |        | UART      | 2 UARTs       | USART3 via STLink/USB or D0(RX)/D1(TX), USART2 on PA3(RX)/PA2(TX) and USART1 on PA10(RX)/PA9(TX) |
 * |        | I2C       | 1 I2C     | |
 * |        | SPI       | 1 SPI     | |
 * |        | USB       | no        | |
 * |        | Timer     | 1 32 bit timer        | |
 *
 * ## Flashing the device
 *
 * The ST Nucleo-F429ZI board includes an on-board ST-LINK V2-1 programmer. The
 * easiest way to program the board is to use OpenOCD. Once you have installed
 * OpenOCD (look [here](https://github.com/RIOT-OS/RIOT/wiki/OpenOCD) for
 * installation instructions), you can flash the board simply by typing
 *
 * ```
 * make BOARD=nucleo144-f429 flash
 * ```
 * and debug via GDB by simply typing
 * ```
 * make BOARD=nucleo144-f429 debug
 * ```
 *
 * ## Supported Toolchains
 *
 * For using the ST Nucleo144-F429 board we strongly recommend the usage of the
 * [GNU Tools for ARM Embedded Processors](https://launchpad.net/gcc-arm-embedded)
 * toolchain.
 */
